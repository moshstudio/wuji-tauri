## Q&A

1. 如何导入默认源？

   打开软件后，如果没有任何源，会提示是否导入`默认订阅源`，确认即可。

2. 是否与其他软件的源兼容?

   不兼容。

3. 如何导入自定义源？

   在电脑版界面中，点击`创建订阅源`，按要求完成所有接口。

4. 订阅源如何更新？

   在`管理订阅源`中更新。
